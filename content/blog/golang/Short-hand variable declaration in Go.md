Variables in Go can have short-hand declarations, like this:

{{< highlight go >}}

name := "Jane"

{{< / highlight >}}

We can also define multiple variables this way, for example:

{{< highlight go >}}

name, surname, age := "Jane", "Smith", 29

{{< / highlight >}}
